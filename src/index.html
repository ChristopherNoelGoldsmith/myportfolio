Make Some Flash Cards

The purpose of this project, is to create a study tool for my wife (she is in school for optometry.) I chose the React and Bootstrap libraries for it's architecture, with the intent of polishing and learning these common libraries. Throughout development of this project I gained a better understanding of hooks and the paradigms CSS has with React.

I decided to use localStorage() API to store the data because of the small scale of its use, and keeping the intent of the project in mind. I intend on intigrating MongoDB as a database for this project in the future and converting it into a fullstack MERN project in the future. For the time being I have different fish to fry.
